

OMERO.HPC
=========

This is a HPC implementation of Weka environment Fast Random Forest that uses a pre-OMERO.fs OMERO server as the front-end.

Requirements
------------

  - RabbitMQ
  - ImageJ

Python packages
---------------
  - Pika

------------
